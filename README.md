# Laba-Laba Sunda

**Laba-Laba Sunda** adalah aplikasi **pemesan makanan khas Sunda** yang dibuat menggunakan **Android Studio (Kotlin)**. Aplikasi ini memudahkan pengguna untuk melihat menu, memilih makanan, dan melakukan pemesanan secara praktis.

---

## Fitur
- Melihat daftar menu makanan khas Sunda.
- Memilih makanan dan langsung melakukan pemesanan.
- Melihat rangkuman pesanan.
- Desain UI sederhana dan user-friendly.
